# L4TM-HyDE Keybindings

This document lists all the keybindings available in L4TM-HyDE. The keybindings are organized by category for easy reference.

## Main Modifier
- `$mainMod` = **Super** (Windows key)

## Power Management

### Session Control
- **Super + Delete** - Exit Hyprland session immediately
- **Super + Backspace** - Launch power menu (wlogout)
- **Super + L** - Lock screen (hyprlock)

### Power Actions (via Power Menu)
- **l** - Lock screen
- **e** - Logout (exit Hyprland)
- **u** - Suspend (lock and suspend)
- **h** - Hibernate
- **s** - Shutdown
- **r** - Reboot

### Direct Power Keybindings
- **Super + Shift + L** - Lock and suspend
- **Super + Shift + H** - Hibernate
- **Super + Ctrl + R** - Reboot
- **Super + Ctrl + S** - Shutdown

## Window Management

### Basic Window Actions
- **Super + Q** - Close focused window
- **Alt + F4** - Close focused window
- **Super + W** - Toggle floating window
- **Super + G** - Toggle window grouping
- **Alt + Return** - Toggle fullscreen
- **Super + Shift + F** - Toggle window pin

### Window Movement
- **Super + Left/Right/Up/Down** - Move focus between windows
- **Alt + Tab** - Move focus down
- **Super + J** - Toggle window split

### Window Resizing
- **Super + Shift + Right** - Resize active window (+30 width)
- **Super + Shift + Left** - Resize active window (-30 width)
- **Super + Shift + Up** - Resize active window (-30 height)
- **Super + Shift + Down** - Resize active window (+30 height)

### Window Positioning
- **Super + Z** - Move window (mouse drag mode)
- **Super + X** - Resize window (mouse resize mode)
- **Super + Shift + Ctrl + Left/Right/Up/Down** - Move active window

## Workspace Management

### Workspace Navigation
- **Super + 1-0** - Switch to workspace 1-10
- **Super + Ctrl + Right** - Switch to next workspace
- **Super + Ctrl + Left** - Switch to previous workspace
- **Super + Ctrl + Down** - Move to first empty workspace
- **Super + Mouse Up/Down** - Scroll through workspaces

### Move Windows to Workspaces
- **Super + Shift + 1-0** - Move focused window to workspace 1-10
- **Super + Ctrl + Alt + Right** - Move window to next workspace
- **Super + Ctrl + Alt + Left** - Move window to previous workspace

### Special Workspaces
- **Super + Alt + S** - Move window to scratchpad
- **Super + S** - Toggle scratchpad workspace

## Application Launchers

### Main Applications
- **Super + T** - Launch terminal (kitty)
- **Super + E** - Launch email client (bluemail)
- **Super + C** - Launch code editor (cursor)
- **Super + F** - Launch file manager (dolphin)
- **Super + B** - Launch web browser (brave)
- **Super + R** - Launch RDP client (remmina)

### Rofi Menus
- **Super + A** - Application launcher
- **Super + Tab** - Window switcher
- **Super + Shift + F** - File explorer (rofi)
- **Super + V** - Clipboard history
- **Super + Shift + V** - Clipboard manager
- **Super + slash** - Keybinds hint

## System Utilities

### Screenshots
- **Super + P** - Partial screenshot capture
- **Super + Ctrl + P** - Partial screenshot (frozen screen)
- **Super + Alt + P** - Monitor screenshot capture
- **Print** - All monitors screenshot capture

### Waybar Customization
- **Super + Shift + B** - Waybar style selector (switch between 10 visual styles)

### System Monitoring
- **Ctrl + Shift + Escape** - System monitor (htop/btop)

### Color Picker
- **Super + Shift + P** - Color picker (copy hex to clipboard)

## Audio Control

### Volume Control
- **XF86AudioMute** - Toggle audio mute
- **XF86AudioMicMute** - Toggle microphone mute
- **XF86AudioLowerVolume** - Decrease volume
- **XF86AudioRaiseVolume** - Increase volume

### Media Control
- **XF86AudioPlay/Pause** - Play/pause media
- **XF86AudioNext** - Next track
- **XF86AudioPrev** - Previous track

## Brightness Control
- **XF86MonBrightnessUp** - Increase brightness
- **XF86MonBrightnessDown** - Decrease brightness

## L4TM-HyDE Specific

### Theme Management
- **Super + Shift + R** - Wallbash mode select menu
- **Super + Shift + T** - Theme select menu
- **Super + Shift + X** - Theme style select menu
- **Super + Shift + W** - Wallpaper select menu

### Waybar Control
- **Ctrl + Alt + W** - Toggle waybar and reload config
- **Super + Alt + Up** - Next waybar mode
- **Super + Alt + Down** - Previous waybar mode

### Wallpaper Control
- **Super + Alt + Right** - Next wallpaper
- **Super + Alt + Left** - Previous wallpaper

### System Utilities
- **Super + K** - Switch keyboard layout
- **Super + Alt + G** - Gamemode (disable hypr effects)
- **Super + Alt + A** - Animations manager

### Game Launcher
- **Alt + Shift + Space** - Game launcher

## Grouped Windows
- **Super + Ctrl + H** - Move to previous window in group
- **Super + Ctrl + L** - Move to next window in group

## Mouse Controls
- **Super + Mouse Left** - Move window
- **Super + Mouse Right** - Resize window
